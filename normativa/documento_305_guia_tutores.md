# Documento 305, guia para tutores

## Uso operativo

Guia institucional para la tarea del tutor de TFDC.

## Contenido comprimido

El documento orienta a tutores sobre acompanamiento, responsabilidades, interaccion con estudiantes y criterios generales de seguimiento. Es relevante para entender expectativas del rol tutor y del proceso de supervision.

## Instruccion para agente

Usar solo como contexto de gestion academica. No contiene pautas tecnicas de biotecnologia.
