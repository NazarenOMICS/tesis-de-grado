# Documento 302 BI, normas de presentacion TFDC

## Uso operativo

Norma principal de formato y presentacion para TFDC de Biotecnologia ORT.

## Contenido comprimido

El documento define pautas especificas para trabajos finales de carrera en Biotecnologia. Cubre estructura formal, presentacion, componentes esperados, criterios graficos y requisitos de entrega. Funciona como referencia normativa para revisar si la tesis cumple formato institucional.

## Instruccion para agente

Consultar antes de cerrar formato, secciones obligatorias, resumen, abstract, caratula y requisitos de presentacion.
