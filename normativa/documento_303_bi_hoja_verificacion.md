# Documento 303 BI, hoja de verificacion

## Uso operativo

Lista de control para verificar cumplimiento de pautas de presentacion.

## Contenido comprimido

El documento funciona como checklist para tribunales o revisores. Reune puntos de formato, estructura y presentacion que deben verificarse en trabajos finales de Biotecnologia. Es util para auditoria final del manuscrito antes de entrega.

## Instruccion para agente

Usar al final de redaccion como control de cumplimiento, no como fuente conceptual.
