# Manual Amersham Low Molecular Weight Calibration Kit for SDS

## Uso operativo

Manual del kit de calibracion de peso molecular para SDS-PAGE.

## Contenido comprimido

El PDF describe el kit Amersham LMW para electroforesis SDS. Contiene informacion del producto, composicion de proteinas marcadoras, uso previsto y condiciones generales para estimar peso molecular en geles. Es soporte tecnico para interpretar o documentar corridas de gel usadas como control de calidad.

## Instruccion para agente

Usar solo para contexto tecnico de geles y marcadores. No contiene resultados del TFDC.
