# Protocolo OD600 EMB version 4

## Uso operativo

Version intermedia del protocolo OD600 para IC50 y MIC de EMB.

## Contenido comprimido

La version 4 enfoca el ensayo en C. glutamicum ATCC 13032, CGXII con sacarosa, lectura OD600 a 16 horas y dos placas con OD600 inicial diferente. Mantiene el objetivo de estimar IC50 y MIC de EMB por microdilucion. Sirve para rastrear decisiones de diseno previas a la version 6.

## Instruccion para agente

Usar para trazabilidad metodologica. Si hay discrepancias de tiempo, densidad inicial o formato de placa, priorizar la version 6.
