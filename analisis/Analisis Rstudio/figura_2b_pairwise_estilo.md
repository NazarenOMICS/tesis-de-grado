# Grafica pairwise estilo figura 2B

## Uso operativo

Salida exploratoria para representar log2FC contra conteo de peptidos como proxy de intensidad.

## Contenido comprimido

El PDF contiene una comparacion pairwise con dos paneles: sin posprocesamiento y con normalizacion por mediana. El eje X representa log2FC y el eje Y representa log10 del conteo de peptidos. Los puntos separan grupos bacterianos, humanos y levadura. Sirve para validar visualmente si el flujo de analisis replica el estilo esperado antes de usar datos definitivos.

## Instruccion para agente

Leer como evidencia de desarrollo de pipeline, no como evidencia biologica primaria.
