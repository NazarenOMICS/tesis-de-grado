# Inbox

Esta carpeta es la bandeja de entrada para material sin ordenar.

Nazareno puede pegar aca archivos, textos, capturas, resultados, tablas, PDFs o notas sin cuidar nombres ni estructura.

Regla operativa: nada dentro de `inbox/` se considera fuente final hasta que un agente lo clasifique, lo mueva o lo copie a la carpeta correspondiente, genere MD operativo si corresponde, actualice `INDEX.md`, actualice `CHANGELOG.md`, haga commit y haga push.

Si un archivo de `inbox/` no se puede categorizar, queda aca y se pide decision minima.
